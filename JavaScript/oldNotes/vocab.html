DATA TYPES - built-in data structures

[STRING] - sequence of text enclosed in single or double quote marks
[NUMBER] - number; no quote marks
[BOOLEAN] - true or false value; no quote marks
[ARRAY] - allows you to store multiple values (NOTE: refer to each member of the array like this: myVariable[0], myVariable[1], etc.)

[OBJECTS] - EVERYTHING IN JS IS AN OBJECT AND CAN BE STORED AS A VARIABLE
